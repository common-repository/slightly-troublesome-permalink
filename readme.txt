This plug-in controls the category in permalink. When the post belongs to two or more categories.

== Description ==

When you assign multiple categories to a post, only one can show up in the permalink. This will be the lowest numbered category. If this plug-in is used, arbitrary categories can be applied to a permalink.

= Some features: =

* Arbitrary categories can be applied to a permalink.
* If a parent category is specified, the child category can be given priority to and applied(optional).
